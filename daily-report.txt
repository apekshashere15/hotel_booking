**Project Overview**

The Luxury Hotel Booking System is a responsive and interactive web-based booking form developed using HTML, CSS, Bootstrap, JavaScript, and jQuery. The project allows users to select travel dates, choose room types, specify guest counts, and calculate booking costs dynamically.

The system focuses on creating a realistic hotel booking interface with modern UI design, form validation, and dynamic pricing features.

---

**Technologies Used**

* **HTML5** – Structure of the booking form
* **CSS3** – Styling, layout, glassmorphism design, and responsiveness
* **Bootstrap** – Responsive grid and UI components
* **JavaScript & jQuery** – Dynamic calculations, validation, and interactivity
* **jQuery UI Datepicker** – Date selection and validation
* **jsPDF** – Receipt PDF generation
* **Git & GitHub** – Version control and project hosting

---

**Key Features Implemented**

* Responsive booking form layout with Bootstrap
* Premium gold themed UI with glassmorphism effect
* Check-in and Check-out date validation
* Guest counter for Adults and Children
* Dynamic room pricing calculation with **18% GST**
* Indian currency formatting (₹) for price display
* Room availability checking with stock limits
* Date overlap validation to prevent duplicate bookings
* Payment method selection (Cash, UPI, Card)
* Automatic **Booking ID generation** for each reservation
* Professional booking confirmation modal with receipt layout
* **Download Receipt (PDF)** functionality
* Loading animation during booking processing
* Automatic form reset after successful booking

---

**Outcome**

The project successfully demonstrates a modern **frontend hotel booking system prototype** with realistic booking features such as dynamic pricing, validation, room availability management, payment selection, booking confirmation, and downloadable receipts.

The system provides a **professional UI/UX experience** while implementing essential frontend development concepts.
